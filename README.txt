# Systems_Optimization_TSN

Requirements:
- Python 3.8

Environment Setup
- Linux
    $ chmod +x setup_env/linux.sh
    $ setup_env/linux.sh
- Windows
    $ setup_env/windows.sh

How to run:
    $ python src\main.py <xml file>
    
- if no arguments are given, the default input is TC5_large1.xml

Output details:
- the generated .xml solution files are placed in the './solution_xml_files/' folder
- naming scheme: <test case file>_solution.xml

Our solutions:
- located inside the 'solution' folder
